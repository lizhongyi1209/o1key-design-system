# o1key — Web UI Kit

Marketing site UI kit for o1key. Pixel-target the system foundations:
- Cream page (`#FAF6EE`) with heavy 1.5–2px ink borders.
- Hard-offset shadows (`3px 3px 0 0`) — the signature.
- Orange `#FF5A1F` is the hero accent. Teal earring-gem is rare.
- Display: Space Grotesk. Body: Inter Tight. Mono: JetBrains Mono.

## Components
- `Nav.jsx` — top nav with wordmark, links, key CTA
- `Hero.jsx` — display hero with mascot frame
- `FeatureGrid.jsx` — 3-up feature cards with hard-offset
- `CodeBlock.jsx` — terminal-style snippet with serial header
- `Pricing.jsx` — three tiers, middle one highlighted
- `Footer.jsx` — minimal mono footer with serial

## Files
- `index.html` — full marketing page composing all components
- Run by opening `index.html` in the preview.
